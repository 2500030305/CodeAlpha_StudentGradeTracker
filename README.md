# CodeAlpha_StudentGradeTracker

## Description
This project is a console-based Java application developed as part of the CodeAlpha Java Programming Internship.

The system allows users to:
- Add student names and marks
- Calculate average marks
- Find highest and lowest scores
- Display student performance report

## Features
- Add multiple students
- Store data using ArrayList
- Calculate average marks
- Display highest and lowest marks
- Console-based user interface

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## How to Run

Compile:
javac StudentGradeTracker.java

Run:
java StudentGradeTracker
